**DESAFIO FRONT-END**

Criar uma página web, (pode ser em html e js puro ou utilizar um framework, fica a seu critério) pra mostrar dados sobre jogos fictícios que devem ser gerados no backend.
Os dados que devem ser mostrados devem ser: Número de gols de ambos os times e probabilidade de vitória de cada um dos times. A combinação de nome dos times para o jogo (Time 1 vs Time 2) e os dados de ambos os times devem ser gerados de forma aleatória pelo backend, em qualquer linguagem, como uma simulação. Os dados devem ser atualizados periodicamente.

Back-end-link: https://simulare-football-game-api.vercel.app/simulate
